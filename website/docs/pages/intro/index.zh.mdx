你将在这里了解到关于 DID Connect 的一切。

## DID Connect 是什么？

DID Connect 是基于 W3C DID 标准设计和实现的用于连接用户和去中心化应用的开放协议和开发套件。

- 开放协议定义了要解决的问题，数据格式和基本流程，详见[这里](/protocol)。
- 开发套件提供了一系列把 DID Connect 集成到应用中以满足不同业务需求的组件，请参考[这里](/developer)。

如果你是开发者，在把 DID Connect 集成到应用之前，请花点时间理解下 DID Connect 的[工作流程](/intro/workflow)。
