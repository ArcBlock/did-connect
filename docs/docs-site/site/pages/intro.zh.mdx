## DID Connect 是什么？

DID Connect 是一个致力于快速创建能够运行在 Blocklet Server 中的 Blocklet 的工具。

Blocklet Server 的文档可以点击这里查看：[Blocklet Server](https://docs.arcblock.io/abtnode/zh/)
